# React.js + Vite.js + TailwindCSS

This is a skeleton template for React + Vite + Tailwind

### Install

```
npx tiged decafdeveloper/template_react-vite-tailwind <your-repo-name>
```

```
cd <your-repo-name>
```

```
npm install
```

```
npm run dev
```
